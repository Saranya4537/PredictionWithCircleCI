This data sets consists of 3 different types of irises’ (Setosa, Versicolour, and Virginica) petal and sepal length, stored in a 150x4 numpy.ndarray
The rows being the samples and the columns being: Sepal Length, Sepal Width, Petal Length and Petal Width.
Outlier imputation is performed on the dataset and then model training is done.
DecisionTreeClassifier is used for model training.
pipeline is created for data processing and model training.
Flask is used to create API to get the inputs from the user and classify the irises based on the inputs.
Github is used as an SCM tool
circle CI is used to deploy the app.
