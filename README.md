**About the dataset:**<br>
This data sets consists of 3 different types of irises’ (Setosa, Versicolour, and Virginica) petal and sepal length, stored in a 150x4 numpy.ndarray
The rows being the samples and the columns being: Sepal Length, Sepal Width, Petal Length and Petal Width.<br>
**exploratory data analysis:**<br>
Outlier imputation is performed on the dataset and then model training is done.<br>
**Model training:**<br>
DecisionTreeClassifier is used for model training.<br>
pipeline is created for data processing and model training.<br>
**User interface API creation:**<br>
Flask is used to create API to get the inputs from the user and classify the irises based on the inputs.<br>
